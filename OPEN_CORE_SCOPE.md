# Rhelma Core Open Source Scope

Rhelma Core is the public developer foundation.

Included:
- Rust core libraries
- Authentication primitives
- Event contracts
- Capability model
- Policy primitives
- Health and telemetry foundations
- Public development tooling
- Documentation and examples

Excluded:
- Product API gateway
- Enterprise administration
- Billing and licensing
- Customer deployment automation
- Production bootstrap scripts
- Tenant provisioning
- Hosted operations
- Commercial AI workflows

The goal is to provide a strong open foundation while preserving commercial platform capabilities.
