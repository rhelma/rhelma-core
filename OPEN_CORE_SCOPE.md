# Rhelma Core Open Source Scope

## Purpose

Rhelma Core provides public foundations for building AI-native workspace systems.

## Included

- Core Rust primitives
- Authentication foundations
- Event contracts
- Capability model
- Policy primitives
- Health and telemetry abstractions
- Developer documentation

## Optional Adapters

- Event transport adapters such as Kafka integrations

## Excluded

Commercial platform capabilities are not included:

- enterprise operations
- hosted services
- billing systems
- customer deployments
- AI operations products
- private integrations
- managed observability
- self-improvement business workflows

The goal is to provide a strong developer foundation while preserving commercial platform capabilities.
