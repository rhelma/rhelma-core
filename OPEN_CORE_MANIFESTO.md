# Rhelma Open Core Manifesto

## What is Rhelma?

Rhelma is a **multi-tenant, multi-region, AI-native platform** built on:

- 🔒 **Zero-trust security** by default
- 🔄 **Event-driven workflows** (Kafka/NATS-ready)
- 📊 **Observability-first** primitives (logs, traces, metrics)
- 🤖 **Self-improvement loops** (proposal → evaluation → approval → apply)

This is the **public, open-source core**. It's useful for:
- Building distributed systems
- Learning modern Rust architecture
- Self-hosting community platforms
- Integration and extension

---

## What's Included (Open Core)

### 🎁 Free & Open
✅ **16 Foundation Crates** — Reusable Rust libraries  
✅ **8 Reference Services** — Working implementations  
✅ **3 SDKs** — JavaScript, Python, Go  
✅ **Complete Documentation** — Architecture, API contracts, examples  
✅ **Local Development Stack** — Docker Compose, scripts  
✅ **MIT OR Apache-2.0 License** — Use commercially  

**Perfect for:**
- Solo developers learning Rhelma
- Teams building self-hosted deployments
- Companies integrating via SDK
- Researchers and academics

---

## What's NOT Included (Commercial)

❌ Hosted SaaS infrastructure  
❌ Advanced admin dashboard  
❌ Enterprise integrations (SAML, LDAP, custom connectors)  
❌ Compliance tooling (SOC2 audit, GDPR automation)  
❌ Commercial AI features (behavioral analytics, predictive models)  
❌ 24/7 priority support  

**These features are available** in **Rhelma Enterprise** for teams that need managed operations and advanced capabilities.

## Business Model: Three Tiers

### Tier 1: Open Core (Free) 👥
For **developers, teams, communities**

- Everything in this repo
- MIT OR Apache-2.0 license
- Self-hosted only
- Community support
- **Pricing:** Free

### Tier 2: Rhelma Community (Free) 🌍
For **community platforms, social networks**

- Built on open core
- Run by community
- Self-hosted or free instance
- Managed by volunteers / sponsors
- **Pricing:** Free + donations

### Tier 3: Rhelma Enterprise (Paid) 🏢
For **organizations, enterprises**

- Everything from Tier 1+2
- **Hosted SaaS** (99.99% SLA)
- **Advanced Admin** (monitoring, RBAC, audit)
- **Enterprise Integrations** (SAML/OAuth, LDAP, webhooks)
- **Compliance** (SOC2, GDPR, HIPAA-ready)
- **Advanced AI** (custom models, behavioral analytics)
- **24/7 Support** (SLA guarantee)
- **Professional Services** (migration, training, custom features)
- **Pricing:** $5k–$50k+/month based on scale
