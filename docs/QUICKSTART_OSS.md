# Rhelma Core OSS Quick Start

## 1. Clone

```bash
git clone https://github.com/rhelma/rhelma-core
cd rhelma-core
```

## 2. Build

```bash
cargo check --workspace
```

## 3. Test

```bash
cargo test --workspace
```

Explore:

- crates/rhelma-core
- crates/rhelma-event
- crates/rhelma-capabilities
