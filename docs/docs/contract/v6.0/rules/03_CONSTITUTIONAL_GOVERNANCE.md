# Constitutional governance (Contract v6.0)

This document is **normative**.

## 1) Scope

Rhelma Realm governance is defined by the **Constitutional Charter** in:

- `docs/governance/CHARTER_v1.md`

Supporting normative docs:

- `docs/governance/GENESIS_AND_SUCCESSION_v1.md`
- `docs/governance/POLICY_BUNDLES_v1.md`

This file exists to anchor constitutional governance within the Contract portal.

## 2) Requirements

1) Nodes and services that consume governance state **MUST** validate Policy Bundles per `POLICY_BUNDLES_v1.md`.
2) Governance actions that affect security, value settlement, or node admission **MUST** be recorded in the Governance Log and MUST be auditable.
3) Emergency actions **MUST** be time-limited and reviewable per the Charter.
