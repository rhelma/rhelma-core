# Rhelma Realm v6 (Rhelma6) — Overview

Rhelma6 is the long-horizon design for making the Rhelma ecosystem **fluid**, **distributed**, and **resilient** — with no single point of control and no single point of failure.

This documentation is **developer-facing**. Normative requirements (MUST/SHOULD/MAY) live in the contract:
