# Governance (Constitutional)

This folder contains Rhelma Realm's **constitutional governance** documents.

These documents define **how power is distributed**, **how policies are signed**, and **how disputes are resolved**, while preserving a **decentralized-by-default** network.

## Start here

1. `docs/governance/CHARTER_v1.md` — Constitutional Charter (normative)
2. `docs/governance/GENESIS_AND_SUCCESSION_v1.md` — Creator identity, succession, recovery
3. `docs/governance/POLICY_BUNDLES_v1.md` — Policy Bundle schema + signing rules

## Relationship to the Contract

- **Technical MUST/SHOULD requirements** live under `docs/contract/v6.0/`.
- This folder defines **constitutional governance** (who can sign what, how disputes resolve, anti-capture guardrails).
