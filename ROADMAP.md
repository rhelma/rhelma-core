# Rhelma Public Roadmap

This roadmap is for public planning. Commercial commitments and customer-specific delivery dates should stay in private planning systems.

## Current Focus

- Stabilize the public Rust workspace around core crates and runnable services.
- Publish clear open-source boundaries and release hygiene.
- Prepare `rhelma.ir` as the official reference site.
- Prepare `asrnegar.ir` as the operational social demo.

## Near Term

- Harden `apps/social-service` for the public demo.
- Document public API contracts and OpenAPI examples.
- Add seed data and smoke tests for the Asrnegar workflow.
- Add public deployment examples that do not include production secrets.
- Improve onboarding docs for local development.

## Later

- Version stable service contracts.
- Publish SDK examples for JavaScript, Python, and Go.
- Add moderation workflow documentation.
- Add demo status and public changelog pages.
- Define long-term support policy for public releases.
